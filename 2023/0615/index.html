Probability, Statistics, Linear Algebra, and Calculus Review

Probability

Sample space (\(S\)): The set of all possible outcomes of an experiment

Event (\(E\)): A subset of the sample space

Intersection: \(A \cap B\), set of outcomes overlapping events A and B

Union: \(A \cup B\), set of outcomes across A and B

Axioms of probability:

  1. \(0 \leq P(E) \leq 1\)
  2. \(P(S) = 1\)
  3. for a set of mutually exclusive events (i.e. \(E_i \cap E_j\) = \(\emptyset\)): \( \bigcup_{i=1}^{n} E_i = \sum_{i=1}^{n} P(E_i) \)

Permutation: given n objects, the number of arrangements of r objects in a given order. \(P(n,r) = \frac{n!}{(n-r)!}\)

Combination: given n objects, the number of arrangements of r objects in any order. \(C(n,r) = \frac{P(n,r)}{r!} = \frac{n!}{r!(n-r)!}\)

Bayes Rule: for events A and B such that \(P(B) > 0 \), \(P(A|B) = \frac{P(A \cap B)}{P(B)} = \frac{P(B|A)P(A)}{P(B)}\) since \(P(A \cap B) = P(A)P(B|A) = P(B)P(B|A)\)

For a partition \({A_i}\) and any event \(B\), then \(P(B) = \sum_{i=1}^{n} P(A_i)P(B|A_i)\) (sum up probabilities comprising B across different sample spaces) and applying Bayes, \(P(A_k|B) = \frac{P(B|A_k)P(A_k)}{\sum_{i=1}^{n}P(B|A_i)P(A_i)}\)

Independence: A and B are independent iff: \(P(A \cap B) = P(A)P(B)\)

Random Variable: a function that maps every element in a sample space to a real line.

Cumulative distribution function (CDF): A monotonically non-decreasing function \(F(x) = P(X \leq x)\). As x approaches negative infinity, F(x) equals 0 and as it approaches positive infinity, F(x) equals 1. Similarly, \( P(a< X \leq b) = F(b) - F(a)\)

Probability density function (PDF): A function f that denotes the probability that X takes on a value between two values the random variable can take, not the probability of that event.

  Discrete Case (i.e. outcome of a coin flip) Continuous Case (i.e. temperature of a room)
CDF \(F\) \(F(x) = \sum P(X = x_{i})\) \(F(x) = \int_{-\infty}^{x} f(y)dy\)
PDF \(f\) [Probability Mass Function]: \(f(x_{j}) = P(X = x_{j}) = p(x_{j})\) \(f(x) = \frac{dF}{dx}\)
Properties \(0\leq f(x_{j}) \leq 1\) and \(\sum_{j} f(x_{j}) =1\) \(f(x) \geq 0\) and \(\int_{-\infty}^{\infty} f(x)dx =1\)

Expected value (\(E[X]\)): Known as the mean or first moment. Found by averaging the results of an event infinitely many times.

Sample Mean: An estimate the true mean by finding the average of a random sample, defined \(\bar{X} = \frac{1}{n} \sum_{i=1}^{n} X_i\).

Central Limit Theorem: Given a random sample with mean \(\mu\) and variance \(\sigma^2\), then \(\bar{X} \underset{n \to +\infty}{\sim} N(\mu, \frac{\sigma}{\sqrt{n}})\)

k-th moment (\(E[X^{k}]\)): Expected value of \(X^{k}\) after infinitely many trials. Here, \(X^{k}\) as \(g(X)\) is used in the general form of the expected value formula. It can also be calculated with the formula \(E[X^k] = i^k \left[\frac{\partial^k \psi}{\partial \omega^k}\right]_{\omega=0}\)

Variance (\(Var(X)\) or \(\sigma^2\)): Measure of the spread of a distribution function. \(Var(X) = E[(X-E[X])^2] = E[X^2]-E[X]^2\)

Sample variance: Used to estimate true variance and given by \(\hat{\sigma}^2 = \frac{1}{n-1} \sum_{i=1}^{n} (X_i - \bar{X})^2\)

Chi-Squared relation: \(\frac{\hat{\sigma}^2(n-1)}{\sigma^2} \sim \chi^2_{n-1}\)

Standard deviation (\(\sigma\)): Measure of spread, compatible with units of actual random variable.

Characteristic function (\(\psi(\omega)\)): The expected value of the complex exponential function \(e^{iX\omega}\).

  Discrete Case Continuous Case
\(E[X]\) \(\sum_{i=1}^{n} x_i f(x_i)\) \(\int_{-\infty}^{+\infty} xf(x)dx\)
\(E[g(X)]\) \(\sum_{i=1}^{n} g(x_i) f(x_i)\) \(\int_{-\infty}^{+\infty} g(x)f(x)dx\)
\(E[X^k]\) \(\sum_{i=1}^{n} x_{ik} f(x_i)\) \(\int_{-\infty}^{+\infty} x_k f(x)dx\)
\(\psi(\omega)\) \(\sum_{i=1}^{n} f(x_i)e^{i\omega x_i}\) \(\int_{-\infty}^{+\infty} f(x)e^{i\omega x} dx\)

Transformation of RVs: If X and Y are related by some function and \(f_X\) and \(f_Y\) denote the PDF of the RVs respectively,the PDF of the transformed random variable Y is given by \(f_Y(y) = f_X(x) \left|\frac{{dy}}{{dx}}\right|\), where the absolute value of the derivative of the transformation function that relates the random variables X and Y. It quantifies the rate of change in Y with respect to X.

Chebyshev's inequality: For a RV X with expected value \(\mu\), and standard deviation \(\sigma > 0, k>0\): \(P(|X-\mu| \geq k\sigma) \leq \frac{1}{k^2}\). This says the probability of X deviating from its mean by a certain number of standard deviations (k\(\sigma\)) or more is bounded by an upper limit equal to the reciprocal of the square of that number.

Joint PDF \(f_{XY}(x_i, y_j) = P(X = x_i \text{ and } Y = y_j)\) \(f_{XY}(x, y) \Delta x \Delta y = P(x \leq X \leq x+\Delta x \text{ and } y \leq Y \leq y+\Delta y)\)

Conditional density \(f_{XY}\): \(f_{X|Y}(x) = \frac{f_Y(y)}{f_{XY}(x, y)}\)

Independence of two RVs: Two random variables X and Y are said to be independent iff \(P(X, Y) = P(X)P(Y) \quad f_{XY}(x, y) = f_X(x) f_Y(y)\)

Covariance: \(Cov(X, Y) \triangleq \sigma_{XY}^2 = E[(X-\mu_X)(Y-\mu_Y)] = E[XY] - \mu_X \mu_Y\)

Correlation: \(Cor(X, Y) = \frac{\sigma_{XY}}{\sigma_X \sigma_Y} = \frac{\sigma_{XY}^2}{\sigma_X^2 \sigma_Y^2}\). \(Cor(X, Y)\) lies in [-1, 1] and is 0 if the RVs are independent.

Statistics

Confidence Level: A confidence interval with confidence level \(1-\alpha\) means \(1-\alpha\) of the time, the true value is in the confidence interval. To find \(CI_{1-\alpha}\) such that \(\text{Pr}(\sigma^2 \in CI_{1-\alpha}) = 1 - \alpha\), calculate the sample variance \(s^2 = \frac{1}{n-1} \sum_{i=1}^{n} (X_i - \bar{X})^2\) then normalize it by finding \(K = \frac{\sigma^2}{s^2(n-1)} \sim \chi^2_{n-1}\), which lies in \(\chi^2\) distribution with n-1 degrees of freedom, then use a look-up table to find \(P(\chi^2_{1} \leq K \leq \chi^2_{2}) = 1 - \alpha\), so in total \(CI_{1-\alpha} = [\frac{s^2(n-1)}{\chi^2_{2}} , \frac{s^2(n-1)}{\chi^2_{1}}]\)

Type I Error (\(\alpha\)): Probability of rejecting the H0 while the H0 is true ("false positive"), expressed \(\alpha = P(T \in R | H_0 \text{ true})\), where T is the test statistic and R is the rejection region to reject the null hypothesis.

Type II Error (\(\beta\)): Probability of not rejecting H0 while it is not true ("false negative"), expressed \(\beta = P(T \notin R | H_0 \text{ not true})\)

P-value: A measure of the strength of evidence against the null hypothesis and quantifies the probability of observing a test statistic as extreme as the one obtained in the sample data. A smaller p-value indicates stronger evidence against the null hypothesis, suggesting that the observed data is unlikely to have occurred by chance alone.

\(\chi^2\) fit (goodness of fit test): Used to assess whether an observed sample of data follows a specific theoretical distribution. The null hypothesis assumes that the observed data follows the specified theoretical distribution. Per bin of data, calculate the expected frequencies expected if the null hypothesis is true. \(T = \sum_{i=1}^{k} \frac{(Y_i - np_i)^2}{np_i} \quad \text{under } H_0 \sim \chi^2_{df}\) with df = (k-1) - #parameters, where k is the number of bins and n is the total number of samples.

Linear modeling: Given \(n\) data points \((x_i, Y_i)\), let \(S_{XY} = \sum_{i=1}^{n}(x_i - \overline{x})(Y_i - \overline{Y}) \quad \text{and} \quad S_{XX} = \sum_{i=1}^{n}(x_i - \overline{x})^2 \quad \text{and} \quad S_{YY} = \sum_{i=1}^{n}(Y_i - \overline{Y})^2\), then estimate coefficients A and B in Y = Ax+B as \(A = \overline{Y} - \frac{S_{XX}}{S_{XY}}\overline{x} \quad \text{and} \quad B = \frac{S_{XY}}{S_{XX}}\)

Correlation coefficient: for two RVs X and Y, \(\rho = \frac{{E[(X - \mu_X)(Y - \mu_Y)]}}{{\sqrt{{E[(X - \mu_X)^2]} \cdot {E[(Y - \mu_Y)^2]}}}}\)

Fisher transformation z-transform: converts the correlation coefficient into a quantity that approximately follows a normal distribution and is defined \( z = \frac{1}{2} \ln\left(\frac{1 - \rho}{1 + \rho}\right) \)

Linear Algebra and Calculus

Vector-vector multiplication:

Inner product: for \(x, y \in \mathbb{R}^n\), \(\mathbf{x}^T \mathbf{y} = \sum_{i=1}^{n} \mathbf{x}_i\mathbf{y}_i \in \mathbb{R}\)

Outer product: for \(x \in \mathbb{R}^m\), \(y \in \mathbb{R}^n\), \(\mathbf{x} \mathbf{y}^T = \begin{bmatrix} x_{1}y_{1} & x_{2}y_{1} & \cdots & x_{m}y_{1} \\ \vdots & \vdots & \ddots & \vdots \\ x_{1}y_{n} & x_{2}y_{n} & \cdots & x_{m}y_{n} \\ \end{bmatrix} \in \mathbb{R}^{m \times n}\)

Matrix-vector multiplication: for \(\mathbf{A} \in \mathbb{R}^{m \times n}, \quad \mathbf{x} \in \mathbb{R}^{n}\), \(\mathbf{Ax} = \sum_{i=1}^{n} \mathbf{a}_{c, i}x_{i} \in \mathbb{R}^{m}\), where \(\mathbf{a}_{c, i}\) is a vector column of A.

Symmetric decomposition: For a symmetric matrix (a square matrix that is equal to its transpose), break the matrix into smaller symmetric rank-one matrix which can be represented as the outer product of a vector with itself, so \( A = \sum_{i=1}^{k} \lambda_i \mathbf{v}_i \mathbf{v}_i^T \), where \(\lambda_i\) represents the eigenvalues of A and \(\mathbf{v}_i\) represents the corresponding eigenvectors. Any matrix A can be broken down into its symmetric and antisymmetric components: \(A = \underbrace{\frac{1}{2}(A + A^T)}_{\text{Symmetric}} + \underbrace{\frac{1}{2}(A - A^T)}_{\text{Antisymmetric}}\)

Norm: A norm (\(\| x \|_p = \left( \sum_{i} |x_i|^p \right)^{\frac{1}{p}}\)) on a vector space \(V\) is a function \(\|\cdot\| : V \rightarrow \mathbb{R}\) that satisfies the following properties:

  1. Non-negativity: \(\|v\| \geq 0\) for all \(v \in V\), and \(\|v\| = 0\) if and only if \(v = 0\) (the zero vector)
  2. Homogeneity/Scalar Multiplication: \(\|\alpha v\| = |\alpha| \|v\|\) for all \(v \in V\) and \(\alpha \in \mathbb{R}\) (or the field of scalars)
  3. Triangle Inequality: \(\|u + v\| \leq \|u\| + \|v\|\) for all \(u, v \in V\)

Positive semidefinite matrix (\(\mathbf{A} \succeq 0\)): Matrix \(\mathbf{A} \in \mathbb{R}^{n \times n}\) is PSD if \(\mathbf{A} = \mathbf{A}^T\) and \(\forall \mathbf{x} \in \mathbb{R}^n, \mathbf{x}^T \mathbf{A} \mathbf{x} \geq 0\)

Positive definite matrix (\(\mathbf{A} \succ 0\)): A PSD matrix that satisfies for all non-zero vector \(\mathbf{x}\), \(\mathbf{x}^T \mathbf{A} \mathbf{x} > 0\).

Eigenvector, eigenvalue: Given a matrix \(\mathbf{A} \in \mathbb{R}^{n \times n}\), \(\lambda\) is said to be an eigenvalue of \(\mathbf{A}\) if there exists a vector \(\mathbf{z} \in \mathbb{R}^n \setminus \{\mathbf{0}\}\), called an eigenvector, such that we have: \(\mathbf{Az} = \lambda \mathbf{z}\). Intuitively, an eigenvector is a vector that remains in the same direction (up to scaling) when a linear transformation or matrix operation is applied to the matrix. The eigenvalue associated with that eigenvector represents the scaling factor by which the eigenvector is stretched or shrunk under the linear transformation.

Spectral Theorem: If \(\mathbf{A}\) is a symmetric matrix, then \(\mathbf{A}\) is diagonalizable by a real orthogonal matrix (a matrix whose transpose is equal to its inverse) \(\mathbf{U} \in \mathbb{R}^{n \times n}\). Let \(\Lambda = \text{diag}(\lambda_1, ..., \lambda_n)\), then \(\mathbf{A} = \mathbf{U} \Lambda \mathbf{U}^T\)

Singular-value decomposition: For a given matrix \(\mathbf{A}\) of dimensions \(m \times n\), the singular-value decomposition (SVD) is a factorization technique that guarantees the existence of \(\mathbf{U}_{m \times m}\) with columns which are orthonormal vectors, \(\Sigma_{m \times n}\) diagonal with singular values in descending order, and \(\mathbf{V}_{n \times n}\) which is an orthogonal matrix, such that: \(\mathbf{A} = \mathbf{U} \Sigma \mathbf{V}^T\)
